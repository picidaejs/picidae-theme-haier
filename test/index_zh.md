-> ## Haier <-

->🍔 一个为项目、开源库、框架建立的通用 Picidae 主题<-

-> _Haier 适合于项目、开源库、框架_<-

-----

-> ### 开始使用 <-

->`npm install picidae-theme-haier -d`<-

```javascript
// picidae.config.js
module.exports = {
  theme: 'picidae-theme-haier'
}
```
